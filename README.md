# m-qiesports-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常作为静态资源托管使用，不依赖特定后端服务，可直接通过浏览器访问。

仓库中的页面内容涉及多种用途，包括但不限于信息展示、工具页面、测试页面等。所有页面均为独立文件，互不依赖。

## 目录说明

```
.
├── index.html          # 仓库首页/导航页（如有）
├── pages/              # 主要页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有归档的 HTML 页面。
- `assets/`：存放页面引用的样式、脚本及媒体文件。

## 页面归档说明

1. 所有页面均为独立 HTML 文件，可直接双击打开或部署到任何静态托管服务。
2. 页面之间的链接、资源引用建议使用相对路径，以确保在不同环境下正常工作。
3. 归档内容不定期更新，新增页面或修改现有页面时会在提交记录中说明。
